Why SharedLearning?

SharedLearning offers you the tools to make your learning experience more efficient and reliable. With new features added with every patch to make your experience even better! Here is some of the reasons on why to use SharedLearning:
  1. Add as many posts as you would like!
  2. Get instant feedback on spelling mistakes when adding new posts.
  3. Translate new words on the go while reading.
  4. Support translation from/to all languages.
  5. Save new words with a simple click to your mini dictionary.
  6. Each post has it's own unique mini dictionary.
  7. Ability to edit translations before saving.
  8. Hidden translations for revision day!

Pre-Alpha release?

This means that the website is a work in progress. It has basic functionalities with basic styling. The aim of this release is to ensure that all the basic functions are working, in addition to fix any bug that users might incounter.
